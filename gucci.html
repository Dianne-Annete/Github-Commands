Github Commands

  • git init
    • -

    Initializing a new Github repository

  • git add "file"
    • -

    Adding a specific file to the staging area

  • git add .
    • -

    Adding all changes in the current directory to the staging area

  • git commit -m "message"
    • -

    Committing the staged changes with a message

  • git branch -M main
    • -

    Renaming the current branch to main

  • git remote add origin "url"
    • -

    Adding a remote repository

  • git push -u origin main
    • -

    Pushes the changes to the remote repository
